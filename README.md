# fluffybutts

## Purpose
Fluffy Butts is a website where someone who is considering adopting a cat can search for available cats via Petfinder. The site also provides users with an assortment of random cat facts and resources that offer access to information and supplies which are commonly needed by new pet owners.

## Built With
* HTML
* CSS
* JavaScript
